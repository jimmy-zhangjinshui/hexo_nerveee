cpap_data_analysis.diagnosis(patient_profiles)¶

Diagnose all patient based on average events and O2 level. Then add the result to their profile.

If average events less than 5.0 and O2 all higher than 93, the diagnosis result is ‘Normal Sleep’. If average events more than 5.0 and O2 all higher than 93, the diagnosis result is ‘Apnea’. If average events less than 5.0 and any O2 lower than 93, the diagnosis result is ‘Hypoxia’. If average events more than 5.0 and any O2 lower than 93, the diagnosis result is ‘Hypoxia Apnea’.

Parameters:
  • name – patient_profiles - patients’ profiles
  • type – list
Returns:

list

cpap_data_analysis.main()¶

Main function of this module. Automatically read the sample data txt file. Create the profile for each patient. Update their profiles with average events, average mask leakage. Then diagnose the patients and add the result to their profile. Finally, store the profiles into separate .json files for each patient.
